# WCM — Watabe-Claude Method (Seismological Application)

## Status

The WCM earthquake application (ToolA/B/C pipeline, JMA catalog analysis)
is currently under active research and peer review.

Full code and datasets will be released **alongside Paper 3**
(Watabe 2026c, in preparation for ESSOAr/EPSL).

## What WCM Does

WCM applies 24-cell polytope geometry to seismic event catalogs
to detect geometric precursors of large earthquakes (M6.6–M9.0).

**Established findings (Papers 1–3):**

- V-transition scaling law: `log₁₀(T_FormA) = 0.470×M − 1.214` (R²=0.994)
- Spatial scaling law: `log₁₀(R) = 0.0352×M − 0.0283`
- min_mag criticality: M<0.8 → V4 direction; M≥0.8 → V21 direction
- FormA false-positive rate: V21 = 0.000% (1,589 windows verified)
- Scale-dependent V-transition paths:
  M9.0 → 5 stages; M7–8 → 4 stages; M6 → 2 stages

**Verified earthquakes:**
2011 Tohoku M9.0, 2016 Kumamoto M7.3, 2018 Hokkaido M6.7,
2016 Tottori M6.6, 2010 Chile M8.8, 2018 Alaska M8.2,
2004 Sumatra M9.1 (retrospective)
